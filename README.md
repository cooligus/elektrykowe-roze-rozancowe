# Discordowy Serwer Różańcowy

Website promoting Discordowy Serwer Różańcowy

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or newer recommended)
- [pnpm](https://pnpm.io/) (or npm/yarn)

### Running the App

Start the development server:

```bash
pnpm dev
# or
npm run dev
# or
yarn dev
```

### Building for Production

```bash
pnpm build
# or
npm run build
# or
yarn build
```
